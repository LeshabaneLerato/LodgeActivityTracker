# Lodge Activity Tracker

## Description
Lodge Activity Tracker is an ASP.NET Core MVC application that manages lodge activities, tracks attendance using QR codes, and allows admins to approve and manage activities.

## Features
- User registration and login
- Admin dashboard
- CRUD operations for activities
- QR code generation for attendance
- Activity approval workflow
- Identity-based authentication

## Technologies Used
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Bootstrap (optional for UI)
- QR code generator (helper class)
